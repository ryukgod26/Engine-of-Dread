# Engine of Dread

This is my First First Person (fpv) game in godot. This is a horror game in which you run from a mysterious creature and if you get caught you will die. This is not the Complete Story It is only Created for the first version of the game.


## About the Project
I created this Project to learn how to make a 3d game in godot. That's why The game is very basic for now. But Io will Improve it in very soon. This is a horror game with red scene. For now the game is very basic and minimal.

## Project Structure

- `scenes/` — All game scenes are in this folder
- `scripts/` — All the game Script (gd script) files are in this folder.
- `assets/` — All the assets like models,Images are in this folder.
